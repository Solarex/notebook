Eclipse
=======
常用快捷键
+ ``ctrl+l`` 跳转行
+ ``ctrl+shift+t`` 打开类型，打开类，接口文件等
+ ``ctrl+shift+r`` 打开资源文件如xml，png等
+ ``ctrl+o`` 快速大纲
+ ``ctrl+q`` 上一个编辑位置
+ ``ctrl+t`` 快速类型层次结构
+ ``ctrl+e`` 快速跳转到另外一个编辑器，会询问，``ctrl+f6`` 同样功能
+ ``ctrl+/`` 注释 ``ctrl+d`` 删除一行
+ ``ctrl+shift+o`` 自动补全 ``ctrl+shift+f`` 格式化
+ ``alt+←`` ``alt+→`` 回退，前进历史记录
+ ``F11`` 调试最后一次执行的程序
+ ``F5`` 跟踪到方法中 ``F6`` 单步执行程序 ``
